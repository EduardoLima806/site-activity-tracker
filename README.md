Site Activity Tracker

RESTfull API scalable to track user activities in site.

Used technologies:

> Java 8
  * Spring Boot
  * Spring Core
  * Spring Security
  * JWT
  * JUnit Tests  
> NodeJs (scalable by fork process)
> Docker
> Docker Compose
> H2 database
> MongoDB
> Redis
> JMeter (Tests Plan for throughput of 1000 req/s with 50 different users)
> Kubernetes (POC example)
